This is the root folder of the LaTeX with Vim with are versioned using
git.

In this folder you will find five subfolders:
    - codes: contain the code of all the examples present;
    - en: reserved for a future english version;
    - figures: contain all the binary figure files;
    - pt: reserved for the portugues version;
    - tables: contain the code of all tables that are not examples.
And the file reference.bib that contain the bibliography used.

All the files are named using '_' instead of ' '.

Files that are included in another file in ther names have, after '@', 
the name of the file in with is included.
